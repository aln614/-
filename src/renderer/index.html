局域网共享

开启后同一局域网设备可以访问这台电脑的生成面板。

可自定义 IP 后面的数字端口，例如 :7861 / :8080 / :9000

本机地址：http://127.0.0.1:7861

局域网地址：http://192.168.110.30:7861

现在切换局域网开关后会立即生效；如果你修改了端口，程序会自动切换到新端口。

局域网用户的 API Key 不会使用主机保存的密钥；他们可以在自己的浏览器本地缓存保存自己的 API Key。

使用说明
