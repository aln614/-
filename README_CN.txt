# LocalApiImageGenerator Electron V8.5 手机端适配版

新增：
- 自动识别 PC / 平板 / 手机访问，同一个局域网/公网地址自动分配界面。
- 手机端底部导航、单列首页生成、两列图片管理、卡片式历史记录、移动端 AI 聊天。
